## 文章格式
```json
{
    "title": "string",
    "author": "string",
    "keywords": "array",
    "description": "string",
    "time": "Date",
    "body": "string-markdown"
}
```

## 新建文章格式
```json
{
    "fileName": "string",
    "title": "string",
    "author": "string",
    "keywords": "array",
    "description": "string",
    "time": "Date",
    "body": "string-markdown"
}
```

## 删除文章格式
```json
{
    "fileName": "string",
    "uuid": "number|string"
}
```
